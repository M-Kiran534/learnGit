git init
-->Initialized empty Git repository in G:/practice/Git_Learn/.git/

git add .
--> '.' refers to adding all the files 

git commit -m "first commit"
-m flag is used for custom messaging

git branch -M main

git remote add origin https://github.com/M-Kiran534/learnGit.git

Push: 
(for 1st push)
git push -u origin main
-u flag is used to link the local repository with the remote repository

(for later push)
git push origin main

Branch Revoke: if branch is present in remote but not in local

git branch -r                     # Check remote branches

git checkout -b main origin/main   # Restore your local 'main' from GitHub

git pull origin main               # Ensure your local 'main' is up to date with GitHub
